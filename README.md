# Customer-churn-prediction
This project predicts whether a customer will leave a company using Machine Learning.

## 🚀 Features
- Data preprocessing
- Model training
- Prediction system

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 📊 Model Used
- Logistic Regression / Random Forest

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run project:
   python main.py
